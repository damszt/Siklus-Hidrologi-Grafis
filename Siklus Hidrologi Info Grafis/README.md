# Siklus Hidrologi – Infografis Interaktif

Web infografis interaktif tentang **Siklus Hidrologi** yang dibangun dengan HTML, CSS, dan JavaScript murni.

## 🚀 Fitur
- **Diagram Siklus Interaktif** – Klik setiap tahapan untuk melihat penjelasan lengkap
- **Animasi Partikel** – Latar belakang partikel air yang bergerak
- **Scroll Animations** – Elemen muncul secara animasi saat di-scroll
- **Mini Quiz** – Uji pemahaman tentang siklus hidrologi (5 soal)
- **Responsif** – Tampil sempurna di desktop, tablet, dan mobile
- **Water Ripple** – Efek riak air saat klik

## 📁 Struktur
```
├── index.html     # Halaman utama
├── style.css      # Desain & animasi
├── app.js         # Logika interaktif
└── vercel.json    # Konfigurasi Vercel
```

## 🛠️ Deploy ke Vercel

### Cara 1: Via Vercel CLI
```bash
npm i -g vercel
vercel
```

### Cara 2: Via GitHub
1. Push repo ini ke GitHub
2. Buka [vercel.com](https://vercel.com)
3. Import repository
4. Klik **Deploy** — selesai!

### Cara 3: Drag & Drop
1. Buka [vercel.com/new](https://vercel.com/new)
2. Seret folder proyek ini ke browser
3. Tunggu deploy selesai
